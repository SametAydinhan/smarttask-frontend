# SmartTask Frontend

SmartTask is a team-focused task management app. This is the frontend built with **Next.js (App Router)**, **TailwindCSS**, and other modern tools.

## 🚧 Technologies
- Next.js 14+
- TypeScript
- TailwindCSS
- Lucide React
- React Hook Form + Zod
- TanStack Query
- Axios
- Next Intl (i18n)

## 📦 Getting Started

```bash
pnpm install
pnpm dev
```
## 📁 Project Structure
- app/: App Router based pages

- components/: UI components

- lib/: Utility helpers

- hooks/: Custom hooks

- services/: API calls (with Axios)
